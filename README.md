# 🌱 AgriGuard

## AI-Powered Crop Disease Detection System

### Overview
AgriGuard helps farmers detect crop diseases using Artificial Intelligence. Farmers can upload a crop leaf image and receive disease predictions, treatment suggestions, and prevention tips instantly.

---

## ✨ Features

- 🔍 Crop Disease Detection
- 🌾 Crop Health Monitoring
- 💊 Treatment Recommendations
- 📊 Farmer Dashboard
- 🌦 Weather Updates
- 🌍 Multi-Language Support

## ⚙️ How It Works

1. Upload a crop image.
2. AI analyzes the image.
3. Disease is detected.
4. Treatment and prevention tips are provided.
